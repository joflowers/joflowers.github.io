A Cheatsheet on HTML Tables

Name Syntax What it does Side Notes
Tables < table > Creates the table structure
Table Rows < tr > Implements table rows
Table Cells < td > Implements the table data cells inside parent elements
Table Headcells < th > Same as table cells except more specified for the use of header content
Scopes < ..scope=row/col > Used to specify whether a table headcell is a row or column
Column Spanning < colspan="int" > Spans out the content column area to 'int' amount of boxes
Row Spanning < rowspan="int" > Spans out the content row area to 'int amount of boxes
Tableheaders < thead > Creates a holder for table header content
Table Bodys < tbody > Creates a holder for table body content
Table Footers < tfoot > Creates a holder for table footer content
